## 1. Visão Geral do Produto

Aplicação web para criação e edição de organogramas organizacionais com interface intuitiva, permitindo que usuários criem, personalizem e exportem estruturas hierárquicas de forma profissional.

* Resolve a necessidade de criar organogramas customizáveis sem software especializado

* Público-alvo: profissionais de RH, gestores, consultores e estudantes

* Valor de mercado: ferramenta acessível e gratuita para visualização de estruturas organizacionais

## 2. Funcionalidades Principais

### 2.1 Papéis de Usuário

| Papel           | Método de Acesso | Permissões Principais                            |
| --------------- | ---------------- | ------------------------------------------------ |
| Usuário Anônimo | Acesso direto    | Criar, editar e exportar organogramas localmente |

### 2.2 Módulos de Funcionalidades

Nossa aplicação de organogramas consiste nas seguintes páginas principais:

1. **Página de Edição**: canvas principal, toolbar de ferramentas, propriedades dos elementos
2. **Página de Exportação**: opções de formato, pré-visualização, configurações de layout
